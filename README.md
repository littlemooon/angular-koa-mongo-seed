angular-koa-mongo-seed
========

seed application using:

	koa
	angular
	mongo
	lodash
	bootstrap

run with:

	'npm install' - install dependencies
	'mongod' - start an instance of mongodb
	'npm start' - start the app on localhost:3000

examples of:

	RESTful CRUD api
	nested angular views
	tabbed angular views
	table view and edit
